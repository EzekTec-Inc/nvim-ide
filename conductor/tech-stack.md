# Tech Stack

## Programming Languages
- **Lua:** The primary language for Neovim configuration and plugin development.
- **Rust:** Supported for high-performance systems programming.
- **Python:** Supported for scripting and data science workflows.
- **C++:** Supported for low-level software engineering.
- **JavaScript/TypeScript:** Supported for web development, specifically the MERN stack.

## Frameworks
- **React.js:** Frontend development framework.
- **Vue.js:** Frontend development framework.
- **Angular:** Frontend development framework.
- **Express.js & Node.js:** Backend API and server runtime environment.
- **Dioxus:** Frontend, Mobile and Backend development framework in Rust programming.
- **Leptos:** Frontend and Backend development framework in Rust programming.

## Databases
- **MongoDB:** NoSQL database for the MERN stack.
- **PostgreSQL:** Relational database for backend development.

## Testing & Development Tools
- **Neotest:** Unified test runner integrated directly into Neovim, supporting Rust (`neotest-rust`) and Python (`neotest-python`).
