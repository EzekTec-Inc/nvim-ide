## Latest Fix Applied

**2026-02-13T23:56:09-07:00:** Fixed Telescope ft_to_lang error in `init.lua`:
```
attempt to call field 'ft_to_lang' (a nil value)
```

The previous metatable proxy approach for the ft_to_lang shim wasn't working correctly because the proxy table was empty and the `__index` metamethod wasn't being triggered in all execution contexts. Fixed by using direct assignment instead of a proxy, with autocmds to re-apply the shim after plugin loading.

**2026-02-13T23:16:52-07:00:** Fixed nvim-ufo E1511 error in `init.lua`:
```
E1511: Wrong number of characters for field "foldopen"
```

The error occurred because Neovim's fold system was trying to use fillchars for foldopen/foldclose. Fixed by setting `vim.o.foldcolumn = "0"` to disable the fold column entirely. nvim-ufo handles fold display via its own virtual text handler.

**2026-02-13T22:08:21-07:00:** Fixed `lua/plugins/treesitter_extended.lua` which was causing the error:
```
module 'nvim-treesitter.configs' not found
```

The file was attempting to require `nvim-treesitter.configs` which no longer exists in nvim-treesitter 1.0+. The file now returns an empty table, as all treesitter configuration is handled in `lua/plugins/treesitter.lua`.
